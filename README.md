# Bitacora Parser

Módulo para parsear `Bitacora.md` en un objeto Kanban.

Uso rápido:

```bash
node examples/runExample.js
```

El parser exporta `parseBitacora(mdContent)` y funciones auxiliares en `src/bitacoraParser.js`.

**Bitacora**

El archivo `Bitacora.md` en la raíz del repositorio contiene el resumen de tareas (pendientes, en desarrollo y completadas). Se actualizó automáticamente.
